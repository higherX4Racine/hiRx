# Information about how the US Dept. of Ed reports College Report Card Data

Information about how the US Dept. of Ed reports College Report Card
Data

## Usage

``` r
COLLEGE_REPORTCARD_API_FIELDS
```

## Format

a data frame with six columns:

- dev-category:

  \<chr\>

- developer-friendly name:

  \<chr\>

- API data type:

  \<chr\>

- VARIABLE NAME:

  \<chr\>

- NAME OF DATA ELEMENT:

  \<chr\>

- NOTES:

  \<chr\>
