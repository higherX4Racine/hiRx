# Metadata about how the US BLS classifies geographic areas

Metadata about how the US BLS classifies geographic areas

## Usage

``` r
BLS_AREA_TYPES
```

## Format

a data frame with three columns:

- prefix:

  \<chr\> the two-letter code for this kind of geograpic area

- digits:

  \<int\> the number of digits this geography needs for its code.

- label:

  \<chr\> a human-readable description of the type of area

## Source

<https://www.bls.gov/cew/classifications/areas/area-guide.htm>
