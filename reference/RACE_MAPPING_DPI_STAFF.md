# Codes that identify race/ethnicity in WI DPI staff demographics tables

The WI DPI is the Wisconsin Department of Public Instruction. It uses
one- letter codes to report the race/ethnicity of staff members.

## Usage

``` r
RACE_MAPPING_DPI_STAFF
```

## Format

a tibble with 2 columns:

- SOURCE RACE:

  The one-letter code for this racial label

- Description:

  The WI DPI's short description of this racial group .

- Race/Ethnicity:

  The broad group for this subgroup in Racine, WI.

## Source

<https://dpi.wi.gov/wise/data-elements/race>
