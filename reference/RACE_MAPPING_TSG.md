# Codes that identify race/ethnicity in tables from Teaching Strategies Gold.

Teaching Strategies LLC, which owns and publishes the GOLD child
assessment suite, has its own set of racial categories that are similar,
but not identical, to the U.S. Census Bureau's categories.

## Usage

``` r
RACE_MAPPING_TSG
```

## Format

a tibble with 3 columns:

- Race:

  The name that Teaching Strategies uses for this group

- Ethnicity:

  Teaching Strategies tracks Hispanic status with this field.

- Race/Ethnicity:

  The broad group for this subgroup in Racine, WI.
