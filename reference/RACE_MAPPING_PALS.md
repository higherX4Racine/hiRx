# Codes that identify race/ethnicity in PALS tables

PALS is the Phonological Awareness and Literacy Screening. It

## Usage

``` r
RACE_MAPPING_PALS
```

## Format

a tibble with 2 columns:

- SOURCE RACE:

  An integer code that corresponds to a race.

- Long Description:

  PALS' verbose description of the racial category.

- Race/Ethnicity:

  The broad group for this subgroup in Racine, WI.
