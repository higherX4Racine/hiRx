# Look up a color from Higher Expectations' palette

Higher Expectations for Racine County has several color palettes that
were designed by the branding team.

## Usage

``` r
higher_ex_cols(...)
```

## Arguments

- ...:

  Character vectors that name colors. Calling without arguments returns
  every color.

## Value

An unnamed vector of colors in hex format
