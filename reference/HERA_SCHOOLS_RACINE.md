# Aesthetic and lookup information for three Higher Eds near Racine, WI.

Aesthetic and lookup information for three Higher Eds near Racine, WI.

## Usage

``` r
HERA_SCHOOLS_RACINE
```

## Format

A tibble with 4 columns:

- OPEID6:

  Integer code for one IHD, which may have multiple campuses

- Name:

  Text name for the Institution of Higher Education

- Color:

  Text representation of the color for plotting each school

- Shape:

  Integer representation of the shape for plotting each school

@source <https://collegescorecard.ed.gov/data/>
